GenerateObject(Type)
Generates an object for a given type. The type needs to be public, have a public default constructor and settable public properties/fields. Currently it supports the following types: Simple types: Int32, String, Enum, DateTime, Uri, etc. Complex types: POCO types. Nullables: Nullable. Arrays: arrays of simple types or complex types.
