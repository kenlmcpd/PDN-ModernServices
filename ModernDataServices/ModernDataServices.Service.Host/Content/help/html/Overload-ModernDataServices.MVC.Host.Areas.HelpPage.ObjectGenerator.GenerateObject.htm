GenerateObject(Type)
Generates an object for a given type. The type needs to be public, have a public default constructor and settable public properties/fields. Currently it supports the following types: Simple types: Int32, String, Enum, DateTime, Uri, etc. Complex types: POCO types. Nullables: Nullable. Arrays: arrays of simple types or complex types. Key value pairs: KeyValuePair Tuples: Tuple, Tuple, etc Dictionaries: IDictionary or anything deriving from IDictionary. Collections: IList, IEnumerable, ICollection, IList, IEnumerable, ICollection or anything deriving from ICollection or IList. Queryables: IQueryable, IQueryable.
